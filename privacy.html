Politica de Privacidade

Produto: SUAP PGD2 PIT/RIT Autofill Pro

Ultima atualizacao: 2026-01-03

Esta politica descreve como a extensao lida com dados ao automatizar o preenchimento de PIT/RIT no SUAP PGD2.

Coleta e uso de dados

A extensao utiliza dados estritamente necessarios para:

  • Executar o preenchimento automatico dos campos na interface do SUAP PGD2.
  • Gerenciar o modo teste (trial) e a ativacao de licenca vinculada ao usuario.

Dados utilizados:

  • Identificador do usuario no SUAP (hash do identificador exibido na interface).
  • Chave de licenca, quando informada pelo usuario.
  • Status de acesso (trial/licenca) e datas de validade.
  • Identificador do dispositivo gerado localmente para controle de sessao.

O que nao fazemos:

  • Não vendemos dados pessoais.
  • Não coletamos conteudo fora do escopo do SUAP PGD2 necessario para a automacao.

Armazenamento e compartilhamento

  • Os dados sao usados apenas para fornecer licenca/trial e a funcionalidade do autofill.
  • O backend de licencas opera em api.proffelipewagner.com.br.
  • Não compartilhamos dados com terceiros para publicidade.

Seguranca

Implementamos medidas razoaveis para proteger dados em transito e em repouso, quando aplicavel.

Retencao e exclusao

Voce pode solicitar a exclusao de dados relacionados a licenca/trial pelo canal de suporte.

Contato

Suporte: GitHub Issues

Isencao de afiliacao

Este produto nao e afiliado, endossado ou mantido pelo SUAP/PGD2. SUAP e usado apenas para indicar compatibilidade.
