Política de Privacidade

Produto: SUAP PGD2 PIT/RIT Autofill Pro

Última atualização: COLE_AQUI_A_DATA (ex.: 2026-01-03)

Esta política descreve como a extensão lida com dados ao automatizar o preenchimento de PIT/RIT no SUAP PGD2.

Coleta e uso de dados

A extensão pode utilizar dados estritamente necessários para:

  • Executar o preenchimento automático dos campos na interface do SUAP PGD2.
  • Gerenciar o modo teste (trial) e a ativação de licença vinculada ao usuário.

O que pode ser coletado (ajuste conforme seu caso):

  • Identificador de licença/usuário (ex.: um ID interno ou e-mail, se você usa isso para ativação).
  • Dados de uso básicos para funcionamento (ex.: status de ativação/trial).

O que NÃO fazemos:

  • Não vendemos dados pessoais.
  • Não coletamos conteúdo de páginas fora do escopo do SUAP PGD2 necessário para a automação.

Armazenamento e compartilhamento

  • Os dados são usados apenas para fornecer o recurso de licença/trial e a funcionalidade do autofill.
  • Não compartilhamos dados com terceiros para fins de publicidade.

Segurança

Implementamos medidas razoáveis para proteger dados em trânsito e em repouso, quando aplicável.

Retenção e exclusão

Você pode solicitar a exclusão de dados relacionados à licença/trial entrando em contato pelo canal de suporte informado abaixo.

Contato

Suporte: GitHub Issues

Isenção de afiliação

Este produto não é afiliado, endossado ou mantido pelo SUAP/PGD2. “SUAP” é utilizado apenas para indicar compatibilidade.
